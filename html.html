Introdução ao HTML

HTML (HyperText Markup Language) é a linguagem padrão para criação de documentos web. HTML (HyperText Markup Language) é uma linguagem de marcação de hipertexto que define a estrutura e o significado do conteúdo da web. É o código que se usa para estruturar uma página web e o seu conteúdo, por exemplo, em parágrafos, listas com marcadores, imagens e tabelas.

Tags HTML Básicas

Tags HTML são instruções que informam ao navegador como formatar o texto. Você pode usar tags para definir itálico, quebras de linha, objetos, bullet points e muito mais. Tags HTML contêm instruções para exibir um elemento na página. Elas começam com um < e terminam com um >. (Por exemplo: < b > A palavra fica em negrito < /b >)

Parágrafos

A tag < p > em HTML é utilizada para definir parágrafos de texto em uma página web. Ela é uma das tags mais simples e importantes para estruturar e organizar conteúdo textual de maneira semântica.

Listas

As tags de listas em HTML são usadas para organizar informações em forma de listas, sejam elas ordenadas ou não. Elas são fundamentais para estruturar conteúdo que necessita de uma sequência específica ou apenas de uma enumeração simples.

Exemplo prático:

  • < ul >: Define a lista não ordenada.
  • < li >: Define cada item da lista
  • < ol >: Define a lista ordenada.

Tabelas

A tag < table > em HTML é utilizada para criar tabelas que organizam dados em linhas e colunas. É uma das estruturas mais fundamentais para exibir informações tabulares de forma organizada e acessível em páginas web.

Exemplo prático:

Imagens

A tag < img > é essencial para a exibição de conteúdo visual em páginas web. Com atributos como src, alt, width, height e outros, você pode controlar como as imagens são exibidas e melhorar a experiência do usuário. É importante usar esses atributos de maneira adequada para garantir que suas imagens sejam acessíveis e se integrem bem ao design geral da página.
